# Christmas Website

[https://bignagpurechristmas-five.vercel.app/](url)

A website to celebrate the Christmas season what I can share with my family. There will be images of the family on the website and a button around each of the images that says "christmasify?". When the button is pressed, the ai image with the christmas theme will pop up, replacing the original image. then after 5 seconds, the button to return the old image will appear and then the button to switch between the two will remain static and the user can switch between the two freely. There should be some snow on the site 

## Features
- Festive design
- Countdown to Christmas (coming soon)
